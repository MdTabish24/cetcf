# CETCF Website - Remaining Tasks

## 1. Payment & Purchase Flow
- **Buy Video Course Flow:** Course detail page -> "Buy Course" -> Razorpay Payment Gateway -> Unlock video dashboard access.
- **Direct RPL Certification Flow:** "Take Exam Directly" -> Payment -> Immediate access to the 100 MCQ Exam portal.

## 2. Live Exam Engine & Certificate Generation
- Connect the MCQ exam page to the database to pull real questions (100 MCQs generated by the AI).
- Implement a 60-minute timer, 50% passing marks logic, and tab-switch anti-cheat feature.
- Generate a Certificate in the database automatically upon passing the exam.

## 3. Database Seeding
- Once the background script finishes generating 225 courses and 22,500 MCQs, run a `seed-mcqs.js` script to insert all questions and courses into the PostgreSQL database for the frontend to access.

## 4. Verification Engine & Results
- Connect the `VerifyPage.tsx` UI to the backend API to verify digital certificates/marksheets using Roll No / Enrollment No.
- Create a new "Results" page where students can view and download their marksheets.

## 5. Multi-Portal Access (Admin & Franchise)
- Route the Partner (Franchise) Lead System form (`PartnerPage`) to the admin dashboard.
- Develop the Admin & Center Portal for managing courses, students, centers, and payments as per Enterprise PRD requirements.
