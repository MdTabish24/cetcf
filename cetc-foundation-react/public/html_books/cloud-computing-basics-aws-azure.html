Cloud Computing Basics — AWS / Azure

📂 Sector: Advanced IT, Programming & Tech Skills 📊 Level: Intermediate ⏱️ Duration: 3 Months 🕐 Hours: 180 💰 Exam Fee: ₹1500
Master cloud computing fundamentals with hands-on training in AWS and Azure platforms. This intermediate-level course equips you with practical skills to deploy, manage, and optimize cloud infrastructure, making you job-ready for India's rapidly growing cloud industry. Gain industry-recognized certifications and launch a lucrative career in cloud computing.

Module 1: Cloud Computing Fundamentals & AWS Basics

📚 Topics Covered

  • Cloud Computing Overview — Definition, models (IaaS, PaaS, SaaS), deployment types (Public, Private, Hybrid)
  • AWS Global Infrastructure — Regions, Availability Zones, Edge Locations, and selecting appropriate regions for Indian compliance
  • AWS Account Setup & Management — Creating AWS account, billing alerts, IAM users, access keys, and security best practices
  • AWS Console Navigation — Understanding dashboard, services menu, resource tagging, and cost management tools
  • Cloud Service Models Comparison — IaaS vs PaaS vs SaaS with real Indian use cases (Flipkart, Paytm, Ola infrastructure)
  • Cloud Benefits & Challenges — Scalability, cost-efficiency, security concerns, and data residency in India (RBI guidelines)
  • AWS Free Tier & Cost Calculator — Estimating costs for common workloads and avoiding unexpected charges
  • Introduction to Azure — Azure regions in India, subscription models, and Azure Portal basics
  • AWS vs Azure Comparison — Feature parity, pricing models, and when to use each platform
  • Cloud Security Fundamentals — Shared responsibility model, encryption, compliance (ISO 27001, SOC 2), and Indian data protection laws

Module 2: AWS Compute Services & EC2 Mastery

📚 Topics Covered

  • EC2 Instances Deep Dive — Instance types (t2, m5, c5, r5), sizing, and selecting appropriate configurations for workloads
  • Launching & Managing EC2 Instances — Console, CLI, and Infrastructure-as-Code approaches using Terraform
  • EC2 Security Groups & Network ACLs — Inbound/outbound rules, port management, and securing instances from unauthorized access
  • Elastic IPs & Load Balancing — Static IPs, Auto Scaling Groups, Application Load Balancer (ALB), and Network Load Balancer (NLB)
  • EC2 Storage Options — EBS volumes, snapshots, IOPS optimization, and cost-effective storage strategies
  • EC2 Monitoring & Logging — CloudWatch metrics, alarms, custom metrics, and troubleshooting common issues
  • Auto Scaling Configuration — Launch templates, scaling policies, and handling traffic spikes for Indian e-commerce platforms
  • Container Services — Docker basics, ECS, EKS introduction, and containerized application deployment
  • Lambda Serverless Computing — Serverless architecture, Lambda functions, triggers, and event-driven applications
  • AWS Systems Manager — Parameter Store, Secrets Manager, and configuration management for production environments
  • Cost Optimization for Compute — Reserved Instances, Savings Plans, Spot Instances, and ROI calculations
  • Hands-on Lab — Deploy multi-tier application with EC2, load balancing, and auto-scaling

Module 3: AWS Storage & Database Services

📚 Topics Covered

  • S3 Fundamentals — Buckets, objects, keys, storage classes, and versioning for data durability
  • S3 Advanced Features — Lifecycle policies, S3 Intelligent-Tiering, Cross-Region Replication (CRR), and data archival
  • S3 Security & Access Control — Bucket policies, ACLs, IAM integration, encryption (SSE-S3, SSE-KMS), and public access prevention
  • S3 Performance Optimization — Multipart uploads, Transfer Acceleration, CloudFront integration, and request rate optimization
  • RDS (Relational Database Service) — Supported engines (MySQL, PostgreSQL, Oracle, SQL Server), instance sizing, and backup strategies
  • RDS High Availability — Multi-AZ deployments, automatic failover, read replicas, and disaster recovery planning
  • RDS Security & Encryption — VPC integration, Security Groups, encryption at rest/transit, and parameter groups
  • DynamoDB NoSQL Database — Tables, items, attributes, capacity modes (on-demand vs provisioned), and GSI/LSI design
  • DynamoDB Optimization — Query patterns, partition key design, throttling prevention, and cost management
  • Database Migration — AWS DMS, homogeneous/heterogeneous migrations, and minimal downtime strategies
  • Backup & Disaster Recovery — Automated backups, point-in-time recovery, snapshot management, and RTO/RPO calculations
  • Hands-on Lab — Design and deploy S3 data lake with lifecycle policies and RDS MySQL database with replication

Module 4: Azure Compute & Storage Services

📚 Topics Covered

  • Azure Virtual Machines (VMs) — Creation, sizing (Standard, Compute, Memory optimized), and lifecycle management
  • Azure VM Networking — Virtual Networks (VNets), subnets, Network Security Groups (NSGs), and routing configuration
  • Azure Load Balancing — Azure Load Balancer, Application Gateway, Traffic Manager, and high availability setup
  • Azure App Service — Web apps, API apps, deployment slots, and continuous deployment with GitHub/Azure DevOps
  • Azure Container Services — Azure Container Instances (ACI), Azure Kubernetes Service (AKS), and container registry
  • Azure Functions — Serverless computing, triggers, bindings, and event-driven architecture on Azure
  • Azure Blob Storage — Storage accounts, containers, blobs, access tiers, and lifecycle management
  • Azure Managed Disks — Types (Standard HDD, Standard SSD, Premium SSD), snapshots, and image creation
  • Azure SQL Database — Elastic pools, backup/restore, geo-replication, and security features (TDE, firewall rules)
  • Azure Cosmos DB — NoSQL global database, consistency models, partitioning, and multi-region replication
  • Azure Storage Security — Shared Access Signatures (SAS), encryption, firewall rules, and private endpoints
  • Hands-on Lab — Deploy multi-tier application on Azure with VMs, App Service, and SQL Database

Module 5: Cloud Security, Networking & Best Practices

📚 Topics Covered

  • IAM & Access Control — AWS IAM policies, roles, users, MFA, and principle of least privilege implementation
  • Azure Active Directory (AAD) — User management, group policies, conditional access, and SSO configuration
  • Network Security — VPCs, subnets, Security Groups, NACLs, WAF (Web Application Firewall), and DDoS protection
  • VPN & Direct Connect — Site-to-Site VPN, AWS Direct Connect, Azure ExpressRoute, and hybrid cloud connectivity
  • Data Encryption — Encryption at rest (KMS, Azure Key Vault), encryption in transit (TLS/SSL), and key management
  • Compliance & Governance — GDPR, ISO 27001, SOC 2, Indian data protection laws, and audit logging
  • AWS CloudTrail & Azure Monitor — API logging, activity tracking, compliance auditing, and security monitoring
  • Incident Response & Disaster Recovery — RTO/RPO planning, backup strategies, failover procedures, and business continuity
  • Cost Optimization Strategies — Reserved Instances, Spot Instances, resource tagging, cost allocation, and budget alerts
  • Performance Optimization — Caching (CloudFront, Redis), CDN usage, database query optimization, and monitoring
  • Well-Architected Framework — AWS pillars (operational excellence, security, reliability, performance, cost), and design patterns
  • Hands-on Lab — Secure multi-tier application with IAM, encryption, VPC, and implement disaster recovery

Module 6: Cloud DevOps, Infrastructure-as-Code & Real-World Projects

📚 Topics Covered

  • Infrastructure-as-Code (IaC) — Terraform, CloudFormation, ARM templates, and version control for infrastructure
  • CI/CD Pipelines — AWS CodePipeline, Azure DevOps, GitHub Actions, and automated deployment workflows
  • Configuration Management — Ansible, Chef, Puppet, and managing infrastructure at scale
  • AWS CloudFormation Templates — Writing YAML/JSON templates, parameters, outputs, and stack management
  • Terraform Basics — HCL syntax, providers, resources, variables, state management, and AWS/Azure modules
  • Monitoring & Logging — CloudWatch, Application Insights, ELK Stack, and centralized log aggregation
  • Alerting & Notifications — SNS, SQS, email alerts, Slack integration, and incident management
  • AWS CodeBuild & CodeDeploy — Automated builds, artifact management, and deployment automation
  • Container Orchestration — Docker Compose, Kubernetes basics, Helm charts, and production deployments
  • Capstone Project 1 — Deploy scalable e-commerce backend on AWS with RDS, S3, CloudFront, and auto-scaling
  • Capstone Project 2 — Deploy microservices application on Azure with AKS, SQL Database, and CI/CD pipeline
  • Capstone Project 3 — Multi-cloud deployment across AWS and Azure with disaster recovery and cost optimization
