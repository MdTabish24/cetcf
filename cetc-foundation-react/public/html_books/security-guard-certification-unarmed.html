Security Guard Certification — Unarmed

📂 Sector: Security & Facility Management 📊 Level: Foundation ⏱️ Duration: 1 Month 🕐 Hours: 60 💰 Exam Fee: ₹1000
Master the essential skills of professional unarmed security work in just 4 weeks. This CETCF-certified course trains you to become a confident, law-abiding security guard capable of protecting people, property, and assets across India's growing security industry. Learn threat assessment, emergency response, and customer service in real-world scenarios.

Module 1: Security Industry Fundamentals & Professional Conduct

📚 Topics Covered

  • Indian Security Industry Overview — growth, regulations, and employment opportunities in metros and tier-2 cities
  • Role & Responsibilities of Unarmed Security Guard — duties in different sectors (residential, commercial, healthcare, retail)
  • Security Guard Act & Legal Framework — Private Security Agencies (Regulation) Act 2005, licensing requirements, compliance
  • Code of Conduct & Professional Ethics — uniform standards, punctuality, confidentiality, and customer service excellence
  • Security Terminology — common terms used in daily operations (perimeter, checkpoint, incident, breach, alert)
  • Workplace Safety & Health Standards — OSHA guidelines adapted for Indian security context, personal hygiene
  • Communication Skills for Security Personnel — clear reporting, radio protocol, written incident documentation
  • Organizational Hierarchy & Chain of Command — reporting structure, supervisor communication, escalation procedures

Module 2: Threat Assessment, Risk Analysis & Situational Awareness

📚 Topics Covered

  • Identifying Security Threats — common threats in Indian premises (theft, trespassing, vandalism, unauthorized access)
  • Risk Assessment Basics — evaluating vulnerability levels, identifying high-risk areas, prioritizing patrol routes
  • Situational Awareness Techniques — observing behavioral patterns, recognizing suspicious activities, maintaining alertness
  • Profiling Without Bias — identifying genuine security concerns while respecting diversity and avoiding discrimination
  • Environmental Scanning — monitoring CCTV feeds, reading body language, noticing environmental changes
  • Threat Levels & Response Protocol — understanding alert levels (green, yellow, orange, red) and corresponding actions
  • Security Vulnerability Mapping — identifying weak points in premises (entry/exit points, blind spots, storage areas)
  • Emergency Recognition — detecting fires, medical emergencies, break-ins, and natural disasters early

Module 3: Access Control, Patrolling & Facility Protection

📚 Topics Covered

  • Access Control Systems — manual gate duty, visitor management, ID verification, entry/exit logging procedures
  • Visitor Management Protocol — checking credentials, issuing passes, escorting visitors, maintaining visitor registers
  • Patrol Techniques — effective foot patrol routes, timing, checkpoint checks, documentation of findings
  • Key & Lock Management — securing keys, maintaining key registers, proper handover procedures, preventing unauthorized access
  • Perimeter Security — checking fencing, gates, windows, doors, identifying tampering, securing vulnerable areas
  • Vehicle & Parking Management — vehicle identification, permit verification, parking supervision, preventing vehicle theft
  • Night Shift Operations — maintaining alertness during night duty, lighting checks, noise monitoring, emergency preparedness
  • Crowd Management — controlling entry during peak hours, maintaining order, preventing congestion, ensuring safety in public areas
  • Asset Protection — monitoring valuable items, preventing theft, maintaining inventory awareness, reporting discrepancies
  • Facility Inspection Rounds — systematic checking of premises, documenting maintenance issues, reporting hazards

Module 4: Conflict De-escalation, Emergency Response & First Aid

📚 Topics Covered

  • Conflict De-escalation Techniques — verbal communication, calm demeanor, active listening, avoiding confrontation escalation
  • Handling Aggressive Behavior — recognizing warning signs, maintaining personal safety, using non-violent tactics, calling for backup
  • Customer Service in Security — balancing firmness with courtesy, addressing complaints, building trust with premises occupants
  • Emergency Response Procedures — fire evacuation, earthquake response, medical emergencies, suspicious package protocols
  • Fire Safety & Evacuation — fire prevention basics, using fire extinguishers (Class A, B, C), evacuation route knowledge, assembly points
  • Basic First Aid — recovery position, CPR basics, treating minor injuries, recognizing serious conditions, calling ambulance (108/102)
  • Medical Emergency Management — assisting injured persons, preventing panic, coordinating with medical responders, documenting incidents
  • Active Shooter/Violent Intruder Response — lockdown procedures, shelter-in-place techniques, evacuation timing, police coordination
  • Incident Reporting & Documentation — writing clear incident reports, photography of scenes, witness statements, maintaining records
  • Crisis Communication — staying calm under pressure, communicating with management, coordinating with emergency services, public reassurance

Module 5: Security Technology, Surveillance & Modern Tools

📚 Topics Covered

  • CCTV System Basics — understanding camera types (dome, bullet, PTZ), monitoring screens, recording systems, footage retention
  • Alarm Systems Operation — motion sensors, door/window sensors, panic buttons, siren systems, responding to false alarms
  • Access Control Technology — key card systems, biometric readers, intercom systems, magnetic locks, troubleshooting common issues
  • Radio Communication — two-way radio protocol, clear communication, emergency codes, radio etiquette, battery management
  • Mobile Security Tools — using security apps, GPS tracking, emergency alert systems, documentation apps, digital reporting
  • Lighting Systems — understanding outdoor lighting, motion-activated lights, emergency lighting, reporting light failures
  • Security Hardware — understanding locks, hinges, door closers, window bars, reinforced entry points, maintenance awareness
  • Technology Troubleshooting — basic problem-solving for common equipment failures, knowing when to call technicians, documenting issues
  • Data Privacy in Security Operations — protecting sensitive information, secure handling of visitor data, GDPR awareness for international clients

Module 6: Health, Safety, Personal Development & Career Advancement

📚 Topics Covered

  • Occupational Health & Safety — understanding health hazards in security work, ergonomics during long shifts, managing fatigue
  • Mental Health & Stress Management — dealing with high-stress situations, recognizing burnout, accessing support resources, sleep management
  • Physical Fitness Standards — basic fitness requirements, maintaining alertness, exercise routines for security guards, nutrition basics
  • Substance Abuse & Workplace Conduct — understanding zero-tolerance policies, recognizing signs of impairment, professional responsibility
  • Career Progression Pathways — advancement to supervisor, team leader, training roles, armed security options, entrepreneurship in security
  • Continuous Learning & Certification — advanced courses (armed security, VIP protection, cybersecurity awareness), skill enhancement opportunities
  • Networking & Professional Development — joining security associations, attending seminars, building industry connections, mentorship
  • Building Personal Brand — developing reputation, gaining recommendations, documenting achievements, preparing for promotions

✅ Eligibility Criteria

  • Minimum age: 18 years (some employers accept 17+ with parental consent)
  • Minimum education: Class 8th pass or equivalent
  • Valid ID proof required (Aadhar, PAN, Voter ID, or Passport)
  • No criminal record (background verification required)
  • Good physical health and fitness (basic medical clearance)
  • Fluency in English or Hindi (ability to read, write, and communicate)
  • Already working as security guard? Apply for Recognition of Prior Learning (RPL) exam
  • No maximum age limit; experience preferred but not mandatory
  • Indian citizenship required for most positions; work visa holders eligible for some roles

🎓 Learning Outcomes

  • Understand Indian security regulations, laws, and professional standards required for unarmed security work
  • Identify security threats, assess risks, and maintain situational awareness in various premises
  • Execute access control procedures, manage visitors, and conduct effective facility patrols
  • De-escalate conflicts using non-violent communication and respond appropriately to emergencies
  • Operate basic security technology including CCTV, alarms, and communication devices
  • Document incidents accurately, communicate professionally, and maintain detailed security records
