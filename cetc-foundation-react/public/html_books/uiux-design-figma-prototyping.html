UI/UX Design — Figma & Prototyping

📂 Sector: Advanced IT, Programming & Tech Skills 📊 Level: Intermediate ⏱️ Duration: 3 Months 🕐 Hours: 180 💰 Exam Fee: ₹1500
Master modern UI/UX design principles and become proficient in Figma, the industry-leading collaborative design tool. This intensive 12-week program equips you with practical skills to create user-centered digital products, interactive prototypes, and design systems used by leading Indian tech companies and startups.

Module 1: Fundamentals of UI/UX Design & Design Thinking

📚 Topics Covered

  • Introduction to UI vs UX — Understanding the distinction and relationship between User Interface and User Experience design in digital products
  • Design Thinking Methodology — Empathize, Define, Ideate, Prototype, Test framework applied to Indian market products
  • User Research Basics — Conducting surveys, interviews, and usability testing with Indian user demographics
  • User Personas & Journey Mapping — Creating realistic personas for Indian audiences (tier-1, tier-2 cities, diverse income groups)
  • Information Architecture (IA) — Organizing content structure for e-commerce, fintech, and social platforms popular in India
  • Design Principles — Gestalt principles, hierarchy, contrast, alignment, and consistency in Indian digital design context
  • Accessibility & Inclusive Design — WCAG guidelines, supporting Hindi/regional languages, color blindness considerations
  • Design Systems & Component Libraries — Building scalable design systems for multi-product organizations
  • Psychology of Colors & Typography — Cultural color meanings in Indian design, font selection for readability across devices
  • Industry Standards & Best Practices — Following design guidelines from Google Material Design and Apple Human Interface
  • Case Study Analysis — Analyzing successful Indian apps (PhonePe, Paytm, Swiggy) and their design decisions

Module 2: Figma Essentials & Interface Mastery

📚 Topics Covered

  • Figma Workspace Setup — Creating projects, teams, and organizing files for professional workflows
  • Navigation & Panels — Mastering the Figma interface, shortcuts, and customizing workspace for efficiency
  • Shapes & Vector Tools — Drawing rectangles, circles, polygons, and using pen tool for custom illustrations
  • Text & Typography — Adding text, managing typography scales, and creating text styles for consistency
  • Layers & Organization — Naming conventions, grouping, and creating component hierarchies for maintainability
  • Colors & Fills — Creating color palettes, applying gradients, managing brand color libraries
  • Strokes & Effects — Adding borders, shadows, blurs, and creating depth in designs
  • Grids & Guides — Setting up 8px grids, column systems for responsive design, and alignment guides
  • Constraints & Resizing — Creating responsive components that adapt to different screen sizes
  • Prototyping Basics in Figma — Linking frames, creating interactive flows, and testing prototypes
  • Collaboration Features — Real-time editing, comments, version history, and sharing designs with stakeholders
  • Plugins & Extensions — Installing and using popular Figma plugins (Unsplash, Iconify, Figma to Code)

Module 3: Wireframing & Low-Fidelity Design

📚 Topics Covered

  • Wireframing Principles — Creating skeletal layouts focusing on structure and functionality
  • Mobile-First Approach — Designing for mobile devices first, then scaling to tablet and desktop (Indian user behavior)
  • Common UI Patterns — Navigation bars, tab systems, card layouts, list views used in Indian apps
  • E-commerce Wireframes — Product listing, detail pages, cart, checkout flows for platforms like Flipkart, Amazon India
  • Fintech Wireframes — Dashboard, transaction history, payment flows for banking and wallet apps (PhonePe, Google Pay)
  • Social App Wireframes — Feed layouts, messaging interfaces, profile screens for apps like Instagram, WhatsApp
  • Forms & Input Design — Text fields, dropdowns, radio buttons, checkboxes, date pickers optimized for Indian use
  • Error States & Empty States — Designing feedback mechanisms and empty states for better user guidance
  • Annotation & Documentation — Adding notes, specifications, and handoff documentation for developers
  • User Flow Diagrams — Creating flowcharts showing user journeys and decision trees
  • Testing Low-Fidelity Prototypes — Quick user testing with wireframes to validate concepts early
  • Iterating Based on Feedback — Refining wireframes based on user and stakeholder input

Module 4: High-Fidelity Design & Visual Design Systems

📚 Topics Covered

  • From Wireframes to Mockups — Elevating wireframes with visual design, colors, and typography
  • Component Creation — Building reusable buttons, cards, modals, and other UI components in Figma
  • Creating Design Tokens — Defining spacing, sizing, color, and typography tokens for consistency
  • Building Design Systems — Structuring components, variants, and documentation for team use
  • Responsive Design — Creating flexible layouts that work across iPhone, Android, tablet, and desktop
  • Micro-interactions — Designing subtle animations, hover states, loading states, and transitions
  • Visual Hierarchy — Using size, color, and positioning to guide user attention effectively
  • Branding & Style Guides — Creating brand guidelines, logo usage, and visual language documentation
  • Accessibility in Design — Color contrast ratios, text sizing, touch target sizes (minimum 44x44px for mobile)
  • Dark Mode Design — Creating dark theme variants for modern Indian apps
  • Localization Considerations — Designing for Hindi, Tamil, Telugu text expansion and RTL considerations
  • Design Handoff — Preparing designs for developer implementation with specifications and assets

Module 5: Interactive Prototyping & Animation

📚 Topics Covered

  • Figma Prototyping Features — Creating interactions, triggers, and actions between frames
  • Navigation Flows — Designing user flows with multiple screens and conditional navigation
  • Micro-interactions — Adding animations to buttons, transitions between screens, and feedback animations
  • Scroll Behaviors — Designing parallax effects, sticky headers, and infinite scroll interactions
  • Gesture Interactions — Designing swipe, tap, drag, and long-press interactions for mobile apps
  • State Management — Designing different states (default, hover, active, disabled, error, loading)
  • Conditional Interactions — Creating branching flows based on user input and decisions
  • Animation Principles — Timing, easing, anticipation, and follow-through in digital animations
  • Prototyping Payment Flows — Designing interactive prototypes for UPI, card, wallet transactions
  • Testing Prototypes with Users — Conducting usability testing with interactive prototypes
  • Performance Optimization — Keeping prototypes lightweight for smooth performance on various devices
  • Presenting Prototypes — Demonstrating prototypes to stakeholders and gathering feedback

Module 6: Portfolio Development & Professional Practice

📚 Topics Covered

  • Project Selection & Curation — Choosing 3-5 best projects that showcase range and skills
  • Case Study Documentation — Writing compelling case studies with problem, process, and solution narratives
  • Before & After Comparisons — Showing design improvements and impact metrics
  • Process Documentation — Documenting research, wireframes, iterations, and final designs
  • Building Portfolio Website — Creating personal portfolio using Figma, Webflow, or WordPress
  • GitHub Integration — Sharing design files, components, and documentation on GitHub
  • Freelance Platforms — Setting up profiles on Upwork, Fiverr, and 99designs for Indian clients
  • Interview Preparation — Discussing design decisions, trade-offs, and handling design critiques
  • Collaboration with Developers — Understanding handoff process, developer constraints, and QA feedback
  • Staying Updated — Following design trends, tools, and best practices in Indian tech industry
  • Networking & Community — Joining design communities, attending meetups, and building professional relationships
  • Freelancing vs Employment — Understanding career options, rates, and growth paths in Indian design market

  • Access to a computer/laptop with minimum 4GB RAM for Figma

  • Understand and apply core UI/UX design principles, design thinking methodology, and user-centered design processes to create effective digital products
  • Master Figma interface, tools, and features to efficiently create wireframes, mockups, components, and interactive prototypes
  • Conduct user research, create personas and journey maps, and apply findings to inform design decisions for Indian user demographics
