CCTV Installation & Maintenance

📂 Sector: Electrical & Electronics Trades 📊 Level: Foundation ⏱️ Duration: 2 Months 🕐 Hours: 120 💰 Exam Fee: ₹1000
Master the essential skills of CCTV system installation, configuration, and maintenance. This hands-on course equips you with practical knowledge to install surveillance systems in homes, shops, offices, and industrial facilities across India. Gain industry-recognized certification and launch a rewarding career in the fast-growing security technology sector.

Module 1: Fundamentals of CCTV Systems & Camera Types

📚 Topics Covered

  • Introduction to CCTV systems — what is CCTV, why it's important, and uses in India
  • Analog cameras (CCD/CMOS) — specifications, resolution (D1, 2MP, 4MP), and applications
  • IP/Network cameras — advantages, PoE technology, and modern surveillance trends
  • Thermal and specialty cameras — night vision, infrared, and PTZ (Pan-Tilt-Zoom) cameras
  • Camera lens types — fixed, varifocal, and motorized lenses with focal length calculations
  • Sensor technology and image quality — frame rates, compression, and video standards
  • Comparison of camera types — analog vs IP, advantages and disadvantages
  • Indian market trends — popular brands (Hikvision, CP Plus, Uniview) and customer preferences
  • Safety and compliance — IS codes for surveillance, privacy laws, and workplace safety

Module 2: Recording Devices, Storage & System Architecture

📚 Topics Covered

  • DVR (Digital Video Recorder) systems — architecture, specifications, and operation
  • NVR (Network Video Recorder) systems — IP-based recording, advantages, and setup
  • Hybrid DVR systems — mixing analog and IP cameras in one system
  • Hard drive selection — capacity calculation, RAID systems, and reliability
  • Storage calculation — determining HDD size based on camera count, resolution, and retention days
  • Video compression standards — H.264, H.265, and MJPEG explained for Indian customers
  • System architecture — star topology, daisy-chain, and network configurations
  • Backup systems — cloud storage, USB backup, and remote access setup
  • Power requirements — UPS sizing, backup battery systems, and power distribution

Module 3: Cabling, Wiring & Electrical Installation Basics

📚 Topics Covered

  • Coaxial cable types — RG59, RG6, and their specifications for analog systems
  • Network cables (Cat5e, Cat6) — twisted pair, shielding, and IP camera wiring
  • Cable routing best practices — conduit selection, cable management, and protection
  • Termination techniques — BNC connectors, RJ45 crimping, and soldering basics
  • Power supply wiring — 12V DC distribution, 230V AC connections, and circuit protection
  • Earthing and grounding — safety standards, lightning protection, and surge protection
  • Cable testing — using multimeter, cable tester, and identifying faults
  • Indoor vs outdoor installation — weatherproofing, conduit types, and UV protection
  • Installation tools — crimper, stripper, tester, multimeter, and safety equipment
  • Common wiring mistakes — troubleshooting poor connections and signal loss
  • IS codes compliance — Indian electrical standards for surveillance installations
  • Cost estimation — material calculation and project budgeting for Indian market

Module 4: Camera Installation, Positioning & Configuration

📚 Topics Covered

  • Site survey and planning — identifying camera locations, coverage areas, and blind spots
  • Angle and field of view calculation — using lens calculator and distance formulas
  • Height and mounting considerations — optimal camera height for different applications
  • Indoor camera installation — ceiling, wall mounting, and dome camera setup
  • Outdoor camera installation — weatherproofing, housing selection, and bracket installation
  • Installation in retail shops — covering billing counter, entry/exit, and stockroom
  • Office and corporate installation — conference rooms, corridors, and parking areas
  • Residential installation — entrance gates, balconies, and living areas
  • Industrial facility installation — factory floors, warehouses, and restricted zones
  • Camera focusing and lens adjustment — manual and motorized lens tuning
  • Image quality verification — checking resolution, color accuracy, and night vision
  • Analog camera configuration — video level adjustment and signal optimization

Module 5: DVR/NVR Setup, Configuration & Remote Access

📚 Topics Covered

  • DVR initial setup — powering on, HDD installation, and system initialization
  • Camera detection and channel mapping — auto-detect and manual channel assignment
  • Recording settings — continuous, scheduled, and motion detection recording
  • Video quality settings — resolution, frame rate, and bitrate configuration
  • Time and date settings — synchronization, NTP server setup, and timezone configuration
  • User account management — admin password, user privileges, and access control
  • Network configuration — IP address, gateway, DNS, and DHCP settings
  • Remote viewing setup — port forwarding, DDNS, and mobile app configuration
  • Cloud integration — connecting to manufacturer cloud services for Indian users
  • Backup and export — manual backup, scheduled backup, and video export procedures
  • Alarm and notification settings — motion alerts, email notifications, and push alerts
  • NVR-specific features — PoE power management, bandwidth optimization, and advanced search

Module 6: Maintenance, Troubleshooting & Customer Support

📚 Topics Covered

  • Preventive maintenance — cleaning lenses, checking connections, and firmware updates
  • Seasonal maintenance — monsoon protection, dust management, and heat dissipation
  • HDD maintenance — SMART monitoring, health check, and replacement procedures
  • Firmware updates — checking versions, downloading updates, and installation process
  • Backup and data recovery — restoring from backup, data corruption handling
  • Common issues diagnosis — no video, poor image quality, and network connectivity problems
  • Camera troubleshooting — checking power, cable continuity, and signal levels
  • DVR/NVR troubleshooting — boot issues, recording failures, and system crashes
  • Network troubleshooting — connectivity issues, IP conflicts, and bandwidth problems
  • Remote access issues — port forwarding, firewall configuration, and DDNS problems
  • Performance optimization — system speed, storage management, and resource allocation
  • Customer communication — explaining issues, providing solutions, and documentation
  • Service documentation — creating installation reports, maintenance logs, and handover documents
  • Warranty and service policies — Indian manufacturer warranty terms and service procedures

✅ Eligibility Criteria

  • Minimum age: 14 years (or 16 years for some states)
  • Minimum education: Class 8th pass (or equivalent)
  • No prior experience required
  • Already working in CCTV field? Apply for Recognition of Prior Learning (RPL) exam
  • Valid government ID proof required (Aadhaar, PAN, Voter ID, or Passport)
  • Applicable for all Indian citizens and registered foreign nationals
  • No age upper limit — open for working professionals and career changers

🎓 Learning Outcomes

  • Install and configure analog and IP CCTV cameras in residential, commercial, and industrial settings
  • Select appropriate cabling, connectors, and tools for CCTV system installation
  • Set up DVR and NVR recording systems with proper storage calculation and backup procedures
  • Configure remote access, mobile viewing, and cloud integration for CCTV systems
  • Diagnose and troubleshoot common CCTV system problems using standard testing methods
  • Perform preventive maintenance and provide professional customer support for CCTV installations
