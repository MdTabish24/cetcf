Network Administration & IT Support

📂 Sector: Advanced IT, Programming & Tech Skills 📊 Level: Intermediate ⏱️ Duration: 3 Months 🕐 Hours: 180 💰 Exam Fee: ₹1500
Master the essential skills of network administration and IT support in this intensive 12-week program designed for the Indian IT industry. Learn to manage networks, troubleshoot systems, and provide technical support while gaining hands-on experience with industry-standard tools and protocols used by Fortune 500 companies operating in India.

Module 1: Networking Fundamentals & OSI Model

📚 Topics Covered

  • Introduction to Networks — LAN, WAN, MAN, and their applications in Indian enterprises
  • OSI Model Layers — Understanding all 7 layers with real-world examples from Indian telecom and banking sectors
  • TCP/IP Protocol Suite — IPv4, IPv6, and subnetting concepts for Indian network infrastructure
  • Network Devices — Routers, switches, hubs, firewalls, and their roles in corporate networks
  • IP Addressing & Subnetting — CIDR notation, subnet masks, and IP allocation strategies
  • Network Topologies — Star, mesh, ring, and hybrid topologies used in Indian data centers
  • Network Cables & Connectors — Ethernet standards (Cat5e, Cat6, Cat6A) and fiber optic basics
  • Network Troubleshooting Tools — Ping, tracert, ipconfig, netstat, and nslookup command-line utilities
  • Introduction to Network Security Basics — Firewalls, NAT, and DMZ concepts
  • Wireless Networking Standards — WiFi 5/6, WPA2/WPA3 security protocols used in Indian offices

Module 2: Windows Server Administration

📚 Topics Covered

  • Windows Server 2019/2022 Installation — Setup, configuration, and role-based deployment
  • Active Directory (AD) Fundamentals — Domain structure, organizational units (OUs), and group policies
  • User & Group Management — Creating user accounts, security groups, and managing permissions in Indian corporate domains
  • Group Policy Objects (GPO) — Implementing security policies, software deployment, and user configurations
  • DHCP Server Configuration — Dynamic IP allocation, scope management, and lease management
  • DNS Server Setup — Zone management, record types (A, MX, CNAME, SRV), and DNS troubleshooting
  • File & Print Server Management — Share permissions, NTFS security, and printer management
  • Windows Server Backup & Recovery — System state backup, disaster recovery planning, and restoration procedures
  • Remote Desktop Services (RDS) — Terminal services, session management, and remote access configuration
  • Windows Server Monitoring — Performance Monitor, Event Viewer, and system logging for IT support teams
  • Patch Management — Windows Update deployment, WSUS configuration, and security patch management
  • Hyper-V Virtualization Basics — Virtual machine creation, snapshots, and resource allocation

Module 3: Linux Server Administration

📚 Topics Covered

  • Linux Operating System Basics — Kernel, shell, and file system hierarchy (FHS) for Indian server environments
  • Linux Installation & Distribution Selection — Ubuntu Server, CentOS, and RHEL in Indian data centers
  • Command Line Interface (CLI) Mastery — Essential commands (ls, cd, mkdir, chmod, chown, grep, sed, awk)
  • File & Directory Permissions — User, group, and other permissions; umask and special permissions (SUID, SGID, sticky bit)
  • User & Group Management in Linux — useradd, usermod, groupadd, and sudoers configuration
  • Package Management — apt/apt-get (Debian/Ubuntu), yum/dnf (RHEL/CentOS), and dependency resolution
  • Linux Networking Configuration — ifconfig, ip commands, network interface configuration, and static/dynamic IP setup
  • SSH (Secure Shell) Configuration — Remote access, key-based authentication, and SSH hardening for security
  • Apache Web Server — Installation, virtual hosts, SSL/TLS certificates, and log management
  • Firewall & Network Security — iptables, firewalld, and UFW configuration for Indian enterprise security policies
  • System Monitoring & Logs — systemd, journalctl, syslog, and performance monitoring with top, htop
  • Cron Jobs & Task Scheduling — Automated backups, maintenance scripts, and scheduled tasks for IT operations

Module 4: Network Security & Troubleshooting

📚 Topics Covered

  • Network Security Principles — CIA triad (Confidentiality, Integrity, Availability) in Indian compliance frameworks
  • Firewall Configuration & Management — Stateful firewalls, ACLs, port filtering, and DDoS protection
  • VPN (Virtual Private Network) Setup — Site-to-site VPN, remote access VPN, and encryption protocols (IPSec, TLS)
  • Intrusion Detection & Prevention — IDS/IPS concepts, Snort/Suricata, and threat detection in Indian networks
  • Network Monitoring Tools — Wireshark packet analysis, tcpdump, and network traffic analysis for troubleshooting
  • Common Network Issues — Connectivity problems, latency, packet loss, and DNS resolution failures
  • Bandwidth Management & QoS — Quality of Service implementation for critical applications in Indian enterprises
  • VLAN Configuration — Virtual LAN segmentation, trunk ports, and inter-VLAN routing
  • Network Troubleshooting Methodology — Systematic approach to identifying and resolving network problems
  • SSL/TLS Certificates — Certificate installation, renewal, and troubleshooting for secure communications
  • Network Redundancy & Failover — Load balancing, failover mechanisms, and high availability architectures
  • Compliance & Security Standards — ISO 27001, GDPR, and RBI guidelines for Indian financial sector networks

Module 5: IT Support & Help Desk Operations

📚 Topics Covered

  • IT Support Fundamentals — Service desk concepts, ITIL basics, and incident management processes
  • Help Desk Ticketing Systems — ServiceNow, Jira, Zoho Desk, and popular Indian BPO ticketing platforms
  • Incident Management — Ticket creation, prioritization, escalation procedures, and SLA management
  • End-User Hardware Support — Desktop troubleshooting, printer issues, peripheral connectivity, and hardware diagnostics
  • Software Installation & Configuration — Application deployment, licensing, and software troubleshooting
  • Remote Support Tools — TeamViewer, AnyDesk, Chrome Remote Desktop, and secure remote assistance protocols
  • User Training & Documentation — Knowledge base creation, user guides, and training delivery for Indian enterprises
  • Email & Collaboration Tools Support — Outlook, Gmail, Microsoft Teams, and Slack troubleshooting
  • VoIP & Telephony Support — Cisco IP phones, Avaya systems, and common telephony issues in Indian call centers
  • Mobile Device Management — BYOD policies, MDM solutions, and mobile security support
  • Customer Communication Skills — Ticket documentation, professional communication, and escalation handling
  • Performance Metrics & Reporting — MTTR (Mean Time To Resolution), ticket closure rates, and SLA compliance reporting

Module 6: Cloud Infrastructure & IT Operations

📚 Topics Covered

  • Cloud Computing Fundamentals — IaaS, PaaS, SaaS models used by Indian enterprises on AWS, Azure, GCP
  • AWS Basics — EC2 instances, S3 storage, RDS databases, and VPC networking for Indian startups and enterprises
  • Microsoft Azure Fundamentals — Virtual machines, App Services, Azure SQL, and hybrid cloud scenarios
  • Google Cloud Platform (GCP) Basics — Compute Engine, Cloud Storage, and BigQuery for Indian data analytics
  • Cloud Security — IAM (Identity & Access Management), encryption, and compliance in Indian cloud deployments
  • Containerization Basics — Docker fundamentals, container images, and container registries for microservices
  • Kubernetes Overview — Orchestration, deployment, and scaling for cloud-native applications
  • Infrastructure as Code (IaC) — Terraform, CloudFormation, and Ansible for infrastructure automation
  • Cloud Backup & Disaster Recovery — RTO/RPO concepts, backup strategies, and failover planning
  • Cost Optimization — Cloud billing, cost monitoring, and optimization strategies for Indian organizations
  • Monitoring & Logging in Cloud — CloudWatch, Stackdriver, and centralized logging for cloud infrastructure
  • Hybrid & Multi-Cloud Architectures — Integrating on-premises and cloud resources for Indian enterprises

✅ Eligibility Criteria

  • Minimum age: 18 years (or 16 years with parental consent)
  • Minimum education: Class 10th pass or equivalent
  • Basic computer literacy required (ability to use Windows/Linux)
  • No prior networking experience required; beginners welcome
  • Already working in IT? Apply for direct RPL (Recognition of Prior Learning) exam
  • Valid government-issued ID proof (Aadhaar, PAN, Driving License, Passport) required at registration
  • Applicable for all Indian citizens and foreign nationals with valid visa/work permit
  • Physical ability to perform hands-on lab work on computers and networking equipment

🎓 Learning Outcomes

  • Configure and troubleshoot enterprise networks using industry-standard protocols and tools
  • Administer Windows Server 2019/2022 with Active Directory, Group Policy, and server roles
  • Manage Linux servers with user/group administration, security hardening, and service management
  • Implement network security measures including firewalls, VPNs, and intrusion detection systems
  • Provide effective IT support through ticketing systems, remote assistance, and user training
  • Deploy and manage cloud infrastructure on AWS, Azure, or GCP platforms with basic DevOps practices
