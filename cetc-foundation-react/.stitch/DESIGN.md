# Design System: CETCF Minimalist Renewal
**Project ID:** 11788332174175301539

## 1. Visual Theme and Atmosphere
- Mood: Minimal, trustworthy, institutional, calm.
- Visual philosophy: low-noise surfaces, precise spacing, restrained elevation.
- Structure source: layout and page content hierarchy follow cetc_website_mockup.html.
- Platform: Web, desktop-first with responsive mobile adaptation.

## 2. Color Palette and Roles
- Page Canvas: Mist Ivory (#f7f7f4)
- Elevated Surface: Pure White (#ffffff)
- Soft Surface: Soft Stone (#efefe9)
- Primary Text: Deep Graphite (#1a1f1c)
- Secondary Text: Slate Moss (#5a625c)
- Border: Light Sage Gray (#d8ddd6)
- Border Strong: Mid Sage Gray (#b9c0b8)
- Primary Action: Forest Slate (#2f5d4f)
- Primary Action Hover: Deep Forest Slate (#23483d)
- Primary Container: Pale Mint Gray (#dde8e2)
- Accent Highlight: Warm Ochre (#d9a441)
- Success State: Grounded Green (#2e7a52)
- Danger State: Terracotta Red (#b44837)

## 3. Typography Rules
- Headline Font: Syne (section titles, hero headings, metric highlights)
- Body Font: DM Sans (paragraphs, table text, labels, controls)
- Label style: uppercase micro-labels with high tracking for metadata and eyebrow text.
- Type rhythm: compact headlines with airy body spacing for readability.

## 4. Component Stylings
- Buttons: pill-shaped, medium weight, low-noise shadows only on primary actions.
- Containers: softly rounded cards (14 to 22 radius) with thin neutral borders.
- Tables: understated striped rows and subtle separators.
- Navigation: sticky translucent top bar with quiet contrast and clear active state.
- Hero: calm two-column editorial block with restrained atmospheric gradients.

## 5. Layout Principles
- Max content width: 1180px.
- Horizontal rhythm: 12 to 24 spacing scale.
- Vertical rhythm: stacked sections with consistent breathing room.
- Responsive behavior: collapse grids at tablet, single-column at mobile.

## 6. Motion
- Single fade-up entrance pattern for homepage blocks.
- Interaction transitions capped to 180ms for clarity-first behavior.
- No ornamental animation loops.
